Créer le fichier service dans : /etc/systemd/system

Activer le service avec : 
sudo systemctl enable handddle-python.service

Démarrer le service :  
sudo systemctl start handddle-python.service

Possibilité de le stopper : 
sudo systemctl stop handddle-python.service


Le service redémarre tout seul s'il y a une erreur


sudo systemctl restart handddle-python.service
